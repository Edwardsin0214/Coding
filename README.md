# Quantitative Finance and Portfolio Analytics Projects

This repository contains my independent research projects in quantitative finance, portfolio analytics, market risk, derivatives pricing, and machine learning. The work mainly focuses on portfolio risk monitoring, regime analysis, relative value strategies, option valuation, and data-driven financial modeling using Python.

## Selected Projects

### 1. Regime-Switching Portfolio Analysis using Hidden Markov Models
Built a regime-based market analysis framework to identify changing market conditions and support risk-aware portfolio allocation.  
The project focuses on drawdown control, volatility management, walk-forward model training, and reducing look-ahead bias in backtesting.

**Key topics:** Hidden Markov Model (HMM), EM algorithm, regime detection, portfolio risk control, backtesting

---

### 2. S&P 500 Risk Dashboard
Developed a market risk dashboard to monitor equity market conditions using indicators such as VIX term structure, yield curve, HY OAS, drawdown, breadth, and rate momentum.  
Designed a composite risk score to summarize market conditions in a practical and visual format for portfolio review.

**Key topics:** portfolio monitoring, market dashboard, risk score, macro indicators, visualization

---

### 3. Cointegration-Based Pair Trading with Dynamic Portfolio Construction
Researched a relative value strategy based on cointegration and mean reversion across paired assets.  
The project evaluates spread behavior, stationarity, and dynamic portfolio construction logic in a systematic trading framework.

**Key topics:** cointegration, mean reversion, unit root test, spread trading, portfolio construction

---

### 4. Credit Risk Modeling with Deep Neural Networks
Built a credit loan default prediction model using deep learning on high-dimensional and imbalanced data.  
Applied feature engineering and preprocessing techniques to improve predictive performance, achieving AUC above 0.82.

**Key topics:** credit risk, classification, deep neural networks, feature engineering, imbalanced data

---

### 5. PCA-Based Stock Selection and Portfolio Analysis
Explored dimensionality reduction techniques for stock selection and portfolio analysis using principal component analysis (PCA).  
The project studies how latent market structure can be used to group securities and support portfolio construction.

**Key topics:** PCA, stock selection, dimensionality reduction, portfolio analysis

---

### 6. Valuation of Quanto Options
Studied the valuation of quanto options and the impact of correlation, FX dynamics, and volatility assumptions on pricing.  
The project focuses on derivatives valuation and the financial intuition behind cross-currency option structures.

**Key topics:** quanto options, derivatives pricing, stochastic modeling, cross-currency valuation
